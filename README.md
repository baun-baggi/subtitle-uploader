# Subtitle Uploader

Firefox extension that allows you to upload subtitles to a video player

To use:
1. Open a webpage that has a video on it.
2. Click on the extension to open the popup.
3. A list of video elements should expand if it's able to detect any video players
4. Select the video element to upload subtitles to.
5. The subtitle widget should appear
6. Select "Browse" to upload a video.
7. Subtitles should now be visible.
8. Tweak the settings in the popup to fix timing, sizing etc.
